AeroSpace Cheatsheet
====================

FOCUS
  alt-h/j/k/l         Focus left/down/up/right

MOVE WINDOW
  alt-shift-h/j/k/l   Move window left/down/up/right

RESIZE
  alt-shift-minus      Shrink window
  alt-shift-equal      Grow window

LAYOUT
  alt-slash            Toggle tiles (horizontal/vertical)
  alt-comma            Toggle accordion layout
  alt-shift-f          Toggle floating/tiling
  alt-ctrl-shift-f     Fullscreen

WORKSPACES
  alt-1/2/3/4          Switch to workspace
  alt-shift-1/2/3/4    Move window to workspace
  alt-tab              Back and forth
  alt-shift-tab        Move workspace to next monitor

APPS (quick launch)
  alt-g                Ghostty
  alt-f                Finder
  alt-o                Obsidian
  alt-m                Mattermost
  alt-s                Safari
  alt-c                Chrome
  alt-t                Telegram
  alt-b                Brain.fm
  alt-e                Final Cut Pro
  alt-z                Zen

SERVICE MODE  (alt-shift-;)
  r                    Reset/flatten layout
  f                    Toggle floating
  backspace            Close all but current
  esc                  Exit service mode
